Fixtures and Expected Results

Fixture 0 — SpO₂→PaO₂ conversion reference points
Purpose: Validate conversion and rounding to 1 decimal.

Input SpO₂ values: [50, 70, 80, 85, 90, 96]
Expected pao2_calc:
	•	50 → 26.9
	•	70 → 36.6
	•	80 → 44.3
	•	85 → 50.0
	•	90 → 58.7
	•	96 → 81.9

Also assert:
	•	SpO₂ < 50 ⇒ pao2_calc is None
	•	SpO₂ > 96 ⇒ pao2_calc is None

Fixture 1 — SpO₂ > 96 yields no PF ratio; acute defaults to 0
Rows:

Expected:

	•	encounter: SOFA_PULM = 0, count_pf_ratio_acute = 0

Fixture 2 — PF ratio computed via 24h FiO₂ day-lookback but excluded from acute due to missing minute-window FiO₂
Rows:
	•	2025-12-31 23:00: FiO₂-only row via low-flow O₂: oxygen_flow_rate=2, is_room_air=False
	•	2026-01-01 00:00: spo2_obs=90, no FiO₂/flow on-row, is_room_air=False

Expected at 00:00 row:
	•	pao2_calc = 58.7
	•	fio2_day_lookback = 27 (2*3 + 21)
	•	fio2_minute_lookback is None
	•	fio2_minute_look_fw is None
	•	fio2_lookback = 27
	•	pf_ratio_temp = 217.41
	•	Does not qualify for acute selection (minute lookback/fw both null)
	•	encounter: SOFA_PULM = 0, count_pf_ratio_acute = 0

Fixture 3 — HFNC produces resp-detail score 3 but Measures caps to 2; 2 PF ratios avoids single-PF suppression
Rows (FiO₂ on-row):
	•	2026-01-01 00:00: spo2_obs=90, support_type=HFNC, fio2_meas=50, oxygen_flow_rate=40, is_room_air=False
	•	2026-01-01 00:30: spo2_obs=85, support_type=HFNC, fio2_meas=50, oxygen_flow_rate=40, is_room_air=False

Expected:

	•	resp-detail sofa_resp_temp = 3 for both
	•	Measures-stage cap (only IMV/NIPPV qualify for 3–4): candidates become 2

	•	encounter: SOFA_PULM = 2

Fixture 4 — IMV allows score 4 with a single PF ratio (no suppression)

Fixture 7 — Baseline selection uses most recent baseline day, not worst-ever
